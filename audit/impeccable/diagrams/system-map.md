## 2. One engine, four runtimes (the no-drift discipline)

The single most relevant Layer-1 pattern: one pure decision core, thin measurement adapters, and a generated in-page bundle so nothing can drift.

```mermaid
flowchart TB
    PURE["<b>Pure core</b>: checkXxx(props)<br/>no DOM access, decision logic only<br/>rules/checks.mjs"]

    PURE --> DOMAD["checkElementXxxDOM(el)<br/>getComputedStyle + getBoundingClientRect"]
    PURE --> OFFAD["checkElementXxx(el, tag, window)<br/>parseFloat(style.width): jsdom has no layout"]

    DOMAD --> R1["Runtime: injected in live browser<br/>(extension + overlay)"]
    DOMAD --> R2["Runtime: Puppeteer headless"]
    OFFAD --> R3["Runtime: jsdom + custom CSS cascade<br/>css-cascade.mjs (~1000 lines, no browser)"]
    PURE --> R4["Runtime: regex on raw source text"]

    R1 -.generated by concatenation from same source.-> NOTE["the live function IS the Node-exported function<br/>cannot drift"]

    style PURE fill:#1a1a2e,color:#eee
    style NOTE fill:#3a2c1c,color:#eee
```

## 3. Live mode: the three-process loop at a glance

Three processes that never share memory. Browser speaks SSE, agent speaks long-poll, server is the only shared state and the only thing that persists.

```mermaid
sequenceDiagram
    participant H as Human
    participant B as Browser overlay
    participant S as Local server
    participant A as Agent
    participant J as Journal (.jsonl)

    A->>S: handshake (spawn server, inject script)
    A->>S: long-poll (park)
    B->>S: open SSE
    S-->>B: connected + "agent listening" beacon
    H->>B: pick element + action + Go
    B->>S: POST event {generate}
    S->>J: append BEFORE acting (durability precondition)
    S-->>A: release parked poll with event
    A->>A: act (Impeccable: write variants; YoinkIt: capture)
    A->>S: POST reply {done}
    S-->>B: SSE update
    H->>B: cycle / tune / Accept
    B->>S: POST event {accept}
    S->>J: append
    S-->>A: release poll
    A->>S: finalize (carbonize / crystallize)
    Note over H,J: kill agent or reload browser anytime →<br/>resume rebuilds state from J and prints the next safe action
```
